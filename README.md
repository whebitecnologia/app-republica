# República de Estudantes - Rateio de Despesas

Aplicação web para cadastro e rateio de despesas em uma república estudantil.

## Funcionalidades
- Cadastro de despesas (nome, valor, responsável, mês, descrição)
- Rateio automático das despesas do mês
- Fechamento mensal: mostra quanto cada um pagou, quanto deve/tem a receber e as transferências sugeridas
- Exportação/importação dos dados em CSV
- Visualização do mês anterior
- Interface moderna com Bootstrap
- Dados salvos no navegador (localStorage)

## Como usar
1. Abra o `index.html` em seu navegador.
2. Cadastre despesas na aba "Despesas".
3. Veja o fechamento do mês e do mês anterior nas abas correspondentes.
4. Exporte ou importe os dados em CSV para backup ou transferência.

## Observações
- Os dados são salvos localmente no navegador. Use a exportação/importação para backup ou troca de computador.
- Não é necessário backend.
